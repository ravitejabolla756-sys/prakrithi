# Prakriti Multicuisine Restaurant Website

A luxurious, modern website built for Prakriti Multicuisine Restaurant using Next.js, Tailwind CSS, and Framer Motion. 

This project is built with **[Next.js](https://nextjs.org/)** and is pre-configured for seamless deployment to **Vercel**.

## 🚀 Easy Deploy to Vercel

The easiest way to deploy your new website is to use the Vercel Platform from the creators of Next.js.

Click the button below to instantly deploy this repository to Vercel:

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fravitejabolla756-sys%2Fprakrithi)

### Manual Deployment via Vercel Dashboard
1. Go to [Vercel](https://vercel.com) and sign in with your GitHub account.
2. Click **Add New** -> **Project**.
3. Import the `ravitejabolla756-sys/prakrithi` repository.
4. Leave all build settings as their defaults (Vercel automatically detects Next.js).
5. Click **Deploy**.

Within 2 minutes, your luxury restaurant website will be live globally!

## 💻 Local Development

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🎨 Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS for utility-first styling
- **Animations**: Framer Motion for smooth, high-end scroll animations
- **Icons**: Lucide React
- **Theme**: Dark Luxury aesthetic with `#D4AF37` Gold accents and glassmorphism components.
